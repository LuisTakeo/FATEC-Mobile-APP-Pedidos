# FATEC-Mobile-APP-Pedidos

Aplicação React Native em Expo para montar um pedido com escolha de produto, bebida, cálculo de total e confirmação com alerta.

Repositório do GitHub: https://github.com/LuisTakeo/FATEC-Mobile-APP-Pedidos

## Como rodar

1. Instale as dependências com `npm install`.
2. Inicie o projeto com `npm start`.
3. Abra no simulador, em um dispositivo físico ou no web, se preferir.
